## 🏛️ SET Index คืออะไร?

**SET = Stock Exchange of Thailand = ตลาดหลักทรัพย์แห่งประเทศไทย**

SET Index = ดัชนีที่บอกภาพรวมว่าตลาดหุ้นไทยเป็นอย่างไร
เหมือน "อุณหภูมิ" ของเศรษฐกิจไทย

ตอนนี้ SET Index อยู่ที่ประมาณ **1,568-1,585 จุด** (มิ.ย. 2026)
- ปีนี้ปรับขึ้น **+24.5% YTD** จากสิ้นปี 2025
- ทำจุดสูงสุดในรอบ 2 ปี 9 เดือน เมื่อเดือน พ.ค. 2026

## 📊 SET50 vs SET100 vs mai คืออะไร?

| ดัชนี | ความหมาย | ใช้ดูอะไร |
|-------|---------|---------|
| SET | ตลาดหลัก (บริษัทใหญ่ทั้งหมด ~500+) | ภาพรวมตลาดไทย |
| SET50 | 50 บริษัทใหญ่ที่สุดและสภาพคล่องสูงสุด | คนลงทุนใช้มากที่สุด |
| SET100 | 100 บริษัทใหญ่ | กระจายมากกว่า SET50 |
| mai | บริษัทขนาดกลาง-เล็ก | High risk, high reward |

**สำหรับเรา → โฟกัส SET50** เพราะนี่คือสิ่งที่กองทุน Index Fund ส่วนใหญ่ติดตาม

---

## 🏢 50 บริษัทใน SET50 คือใคร?

บริษัทที่คุณรู้จักแน่ๆ อยู่ใน SET50 ทั้งหมด:
- **ธนาคาร:** KBANK, SCB, KTB, BBL, BAY
- **พลังงาน:** PTT, PTTEP, PTTGC, IRPC
- **ค้าปลีก:** CPALL (7-Eleven), MAKRO, BJC
- **สื่อสาร:** ADVANC (AIS), TRUE, DTAC
- **โรงพยาบาล:** BDMS (กรุงเทพดุสิต), BH
- **อสังหา:** CPN (Central Pattana), LH

ลงทุนใน SET50 = ลงทุนในบริษัทใหญ่ๆ ของไทยทั้ง 50 บริษัทพร้อมกัน
