## 📖 เรียนรู้อะไรวันนี้ (5 นาที)

**Pivot Table คืออะไร?**

ลองนึกภาพว่าเรามีรายการซื้อขายหุ้น 200 รายการในเดือนเดียว — เราอยากรู้ว่า "แต่ละตัวทำกำไรไปเท่าไหร่?" โดยไม่ต้องนั่งบวกทีละแถว

Pivot Table คือเครื่องมือใน Excel ที่ช่วยให้เรา **สรุปข้อมูลชุดใหญ่ได้ในไม่กี่วินาที** แค่ลาก-วาง ไม่ต้องเขียนสูตรซับซ้อน

**ทำไมต้องรู้?**
- นักวิเคราะห์การเงินทุกคนใช้มัน — เป็น skill ที่บริษัทมองหา
- ช่วยดูพอร์ตลงทุนตัวเองได้เร็วขึ้น เช่น "เดือนนี้ขาดทุนจากหุ้นกลุ่มไหน?"
- ต่อยอดไป Power BI และ SQL ได้ง่ายขึ้นในอนาคต

## ⚡ ลองทำเลย (15 นาที)

เปิด Excel ขึ้นมา แล้วพิมพ์ข้อมูลนี้ลงไป:

| วันที่ | หุ้น | ประเภท | จำนวน (บาท) |
|--------|------|--------|-------------|
| 1/6/69 | XAU/USD | ซื้อ | 5000 |
| 5/6/69 | BTC | ซื้อ | 3000 |
| 10/6/69 | XAU/USD | ขาย | 5500 |
| 15/6/69 | SET ETF | ซื้อ | 2000 |
| 20/6/69 | BTC | ขาย | 3200 |
| 25/6/69 | SET ETF | ขาย | 1900 |

**ขั้นตอน:**

1. พิมพ์ข้อมูลด้านบนลงใน Excel (row 1 = หัวตาราง)
2. คลิกที่ cell ไหนก็ได้ในตาราง
3. ไปที่เมนู **Insert → PivotTable** → กด OK
4. ทางขวามือจะมีกล่อง "PivotTable Fields" — ลาก:
   - **หุ้น** ไปวางที่ช่อง **Rows**
   - **จำนวน (บาท)** ไปวางที่ช่อง **Values**
5. ดูผลลัพธ์ — Excel จะสรุปให้ทันทีว่าแต่ละ asset มียอดรวมเท่าไหร่

**คำถามให้ลองคิด:** ดูจาก Pivot Table แล้ว — BTC กำไรหรือขาดทุน? 😄
