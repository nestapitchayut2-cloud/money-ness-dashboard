## 📖 เรียนรู้อะไรวันนี้ (5 นาที)

**Power Query = เครื่องมือดึงข้อมูล + ทำความสะอาดข้อมูล แบบไม่ต้องเขียนสูตร** อยู่ใน Excel อยู่แล้ว (แท็บ **Data** → กลุ่ม **Get & Transform Data**) ไม่ต้องติดตั้งอะไรเพิ่ม

ทำไมต้องรู้:
- Pivot Table เก่งเรื่อง "สรุป" ข้อมูล แต่ถ้าข้อมูลตั้งต้นรก (หัวตารางเพี้ยน, มีแถวว่าง, ตัวเลขติด text) Pivot จะพังหรือสรุปผิด — Power Query คือขั้นตอนก่อนหน้านั้นที่จัดข้อมูลให้เรียบร้อยก่อน
- นักวิเคราะห์ใช้เวลา ~80% ไปกับการทำความสะอาดข้อมูล — Power Query ช่วยลดงานส่วนนี้ทั้งหมด
- จุดเด็ดที่สุด: ทำ "ขั้นตอน" ไว้ครั้งเดียว (เช่น import + ลบคอลัมน์ + เปลี่ยน format) แล้วกด **Refresh** ทีหลัง มันจะทำซ้ำขั้นตอนเดิมกับข้อมูลใหม่ให้อัตโนมัติ — ไม่ต้องทำมือใหม่ทุกเดือน
- นี่คือ concept เดียวกับที่จะใช้ตอนต่อข้อมูลเข้า Power BI ใน Phase 2

**concept สำคัญ:** Power Query ไม่ใช่การ "แก้ข้อมูลตรงๆ" แต่คือการบันทึก "สูตรทำความสะอาด" ไว้เป็นขั้นๆ (เรียกว่า Applied Steps) แล้วรันซ้ำได้ทุกครั้งที่ข้อมูลเปลี่ยน

## ⚡ ลองทำเลย (15 นาที)

ใช้ไฟล์ที่มีอยู่แล้ว: `Investment_Portfolio_v3.xlsx`

1. เปิดไฟล์ → เลือกตารางข้อมูลพอร์ต (ถ้ายังไม่ใช่ Table ให้กด **Ctrl+T** แปลงเป็น Table ก่อน)
2. ไปแท็บ **Data** → กด **From Table/Range** (อยู่ในกลุ่ม Get & Transform Data ซ้ายสุด)
3. หน้าต่าง **Power Query Editor** จะเปิดขึ้นมา — จะเห็นข้อมูลเหมือนเดิม แต่มี panel **Applied Steps** อยู่ขวามือ
4. ลองทำความสะอาด 2 อย่าง:
   - คลิกขวาที่ชื่อหัวคอลัมน์ใดก็ได้ → **Rename** เปลี่ยนชื่อคอลัมน์
   - คลิกที่ไอคอนเล็กๆ มุมซ้ายของหัวคอลัมน์ตัวเลข (เช่น ABC หรือ 123) → เปลี่ยน Data Type เป็น **Currency** หรือ **Whole Number**
5. สังเกต panel Applied Steps ด้านขวา — จะเห็นว่าทุกอย่างที่ทำถูกบันทึกเป็น step ไว้ทั้งหมด (นี่คือจุดต่างจาก copy-paste)
6. กด **Close & Load** (มุมซ้ายบน) → ข้อมูลที่คลีนแล้วจะเด้งกลับมาเป็นตารางใหม่ใน Excel
7. โบนัสทดสอบพลังจริง: กลับไปแก้ตัวเลขในตารางต้นฉบับ (ชีทเดิม) → ไปแท็บ **Data** → กด **Refresh All** → ดูว่าตารางที่ import มาอัปเดตตามอัตโนมัติไหม โดยไม่ต้องทำ step ซ้ำเลย

**คำถามให้คิด:** ถ้าทุกเดือนต้องอัปเดตพอร์ตตัวเองในไฟล์ Excel — Power Query ช่วยประหยัดเวลาตรงไหนเทียบกับ copy-paste มือ?

**Vocab วันนี้** (จดลง `ENGLISH/vocab-list.md`):
- *transform* = แปลง/ปรับเปลี่ยน (ข้อมูล) → "Power Query lets you transform messy data automatically."
- *refresh* = รีเฟรช/อัปเดตข้อมูลใหม่ → "Click Refresh All to update the report with new data."
- *applied step* = ขั้นตอนที่บันทึกไว้ → "Every action you take becomes an applied step you can undo."
