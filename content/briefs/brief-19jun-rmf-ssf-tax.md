# 📅 พฤหัสบดี 19 มิ.ย. — ลดภาษีได้หลักหมื่น: RMF + Thai ESG คนทำงานประจำต้องรู้
*อ่านได้ใน 15-20 นาที | Layer: 🟢 50% Stable*
*✅ Fact-Checked: มิ.ย. 2026 | แหล่ง: Kasikorn Asset, Finnomena, Bangkok Bank, Krungsri*

## เริ่มจากพื้นฐาน — ภาษีเงินได้ไทยทำงานยังไง

ไทยใช้ระบบ "ภาษีเงินได้แบบก้าวหน้า" (Progressive Tax) — ยิ่งได้มาก ยิ่งเสียภาษีมาก

**อัตราภาษีเงินได้บุคคลธรรมดา (ปี 2026):**

| เงินได้สุทธิ | อัตราภาษี |
|-------------|----------|
| 0 – 150,000 บาท | 0% |
| 150,001 – 300,000 บาท | 5% |
| 300,001 – 500,000 บาท | 10% |
| 500,001 – 750,000 บาท | 15% |
| 750,001 – 1,000,000 บาท | 20% |
| 1,000,001 – 2,000,000 บาท | 25% |
| 2,000,001 – 5,000,000 บาท | 30% |
| 5,000,001 บาทขึ้นไป | 35% |

**สูตรคำนวณเงินได้สุทธิ:**
```
เงินได้พึงประเมิน (เงินเดือน + bonus + commission ทั้งปี)
- ค่าใช้จ่าย (50% แต่ไม่เกิน 100,000 บาท)
- ค่าลดหย่อนส่วนตัว (60,000 บาท)
- RMF / Thai ESG / ประกัน ฯลฯ
= เงินได้สุทธิ → เอาไปคำนวณภาษีตามตาราง
```

## เครื่องมือลดหย่อนภาษีที่ใช้ได้ตอนนี้

> ⚠️ **SSF (Super Saving Fund) ยกเลิกแล้ว** — Thai ESG เข้ามาแทนที่

### 1. RMF (Retirement Mutual Fund) — กองทุนเพื่อเกษียณ

| รายละเอียด | ข้อมูล |
|-----------|--------|
| ลดหย่อนได้สูงสุด | 30% ของรายได้ ≤ **500,000 บาท** |
| เงื่อนไขถอน | อายุ **55 ปีขึ้นไป** + ถือมาแล้ว **5 ปีขึ้นไป** |
| ต้องซื้อต่อเนื่องไหม | ✅ ต้องซื้อทุกปี (เว้นได้ 1 ปีเท่านั้น ถ้าเว้น 2 ปีติด = ผิดเงื่อนไข ต้องคืนภาษีทั้งหมด) |
| นับรวมกับ | PVD + กอช. + ประกันบำนาญ รวมกัน ≤ 500,000 |

### 2. Thai ESG — กองทุนรวมไทยเพื่อความยั่งยืน (ESG)

| รายละเอียด | ข้อมูล |
|-----------|--------|
| ลดหย่อนได้สูงสุด | 30% ของรายได้ ≤ **300,000 บาท** |
| เงื่อนไขถอน | ถือครบ **5 ปีนับวันชนวัน** แล้วขายได้เลย (ไม่ต้องรออายุ 55) |
| ต้องซื้อต่อเนื่องไหม | ❌ ไม่บังคับ ซื้อปีไหนก็ได้ ยืดหยุ่นกว่า RMF |
| นับรวมกับ RMF ไหม | **❌ ไม่นับรวม — วงเงินแยกต่างหากจาก 500K** |
| ช่วงเวลา | โปรแกรม 1 ม.ค. 2567 – 31 ธ.ค. 2569 |

## 💡 ข้อสำคัญที่สุด — Thai ESG วงเงินแยกจาก RMF

```
RMF + PVD + กอช. + ประกันบำนาญ ≤ 500,000 บาท (กลุ่ม 1)
            +
Thai ESG ≤ 300,000 บาท (กลุ่ม 2 — แยกต่างหากเลย)

รวมกันลดหย่อนได้สูงสุด: 800,000 บาท ‼️
```

นี่คือสิ่งที่คนส่วนใหญ่ไม่รู้ และทำให้เสียสิทธิ์ไปฟรีๆ

## คำนวณจริง — ประหยัดได้เท่าไหร่กัน

### Scenario A: Sales Rep รายได้ 600,000 บาท/ปี

**ภาษีก่อนลดหย่อน:**
```
รายได้:         600,000 บาท
หักค่าใช้จ่าย:  -100,000 บาท
หักลดหย่อนตัว:  -60,000 บาท
เงินได้สุทธิ:   440,000 บาท

ภาษี: 0 + 7,500 + 14,000 = 21,500 บาท
```

**ซื้อ RMF 100,000 บาท:**
```
เงินได้สุทธิ: 440,000 - 100,000 = 340,000 บาท
ภาษี: 0 + 7,500 + 4,000 = 11,500 บาท
💰 ประหยัด: 10,000 บาท
```

**ซื้อ RMF 100,000 + Thai ESG 100,000 = 200,000 บาท:**
```
เงินได้สุทธิ: 440,000 - 200,000 = 240,000 บาท
ภาษี: 0 + 4,500 = 4,500 บาท
💰 ประหยัด: 17,000 บาท
```

---

### Scenario B: Sales Rep + Commission รายได้ 1,000,000 บาท/ปี

**ภาษีก่อนลดหย่อน:**
```
รายได้:        1,000,000 บาท
หักค่าใช้จ่าย:   -100,000 บาท
หักลดหย่อนตัว:   -60,000 บาท
เงินได้สุทธิ:    840,000 บาท

ภาษี: 0 + 7,500 + 20,000 + 37,500 + 18,000 = 83,000 บาท
```

**ซื้อ RMF 300,000 + Thai ESG 300,000 = 600,000 บาท:**
```
เงินได้สุทธิ: 840,000 - 600,000 = 240,000 บาท
ภาษี: 0 + 4,500 = 4,500 บาท
💰 ประหยัด: 83,000 - 4,500 = 78,500 บาท ‼️
```

ซื้อแค่ 600K ประหยัดได้เกือบ 80K — **นั่นคือผลตอบแทนทันทีกว่า 13% ก่อนที่กองทุนจะขยับเลย**

## กลยุทธ์สำหรับ Ness

**1. คำนวณก่อนซื้อเสมอ** — ดูว่า marginal rate ปีนี้อยู่ที่กี่ % แล้วซื้อให้ประหยัดสูงสุด

**2. จัดสรรระหว่าง RMF กับ Thai ESG**
- RMF = เงินที่ lock ยาวสุดจนอายุ 55 → ใช้กับส่วนที่แน่ใจว่าไม่ต้องแตะ
- Thai ESG = ยืดหยุ่นกว่า ถือ 5 ปีแล้วขายได้ → เหมาะถ้าอาจต้องใช้ก่อนอายุ 55

**3. RMF ซื้อกระจายตลอดปี** ไม่ต้องรอ ธ.ค. (Thai ESG ก็เช่นกัน)

**4. เลือกกองทุนที่ match portfolio**
- RMF → ตัวที่ track S&P500 หรือ All-World Index (TER ≤ 0.5%)
- Thai ESG → ต้องเป็น ESG-screened fund ตามนิยาม ดูรายชื่อใน set.or.th/thaiesg

**5. ภาษีที่ประหยัดได้ → ลงทุนต่อทันที**
- ประหยัดภาษีได้ เดือน ม.ค. ปีถัดไป → เอาไป DCA 35% Growth layer ต่อ

## โยงกับ Canada Plan

ถ้า Ness ไป Canada 2-3 ปี → เงิน RMF และ Thai ESG ที่ซื้อไว้แล้วยังอยู่ในกองทุน ไม่หาย
แต่ถ้าปีที่ไป Canada ไม่ได้ซื้อ RMF → เว้นได้ 1 ปีโดยไม่ผิดเงื่อนไข

ถ้าไป Canada 2 ปีติด → **ต้องวางแผนซื้อ RMF ก่อนออกเดินทาง** ไม่งั้นผิดเงื่อนไขและต้องคืนภาษีทั้งหมด

## สรุป — ถ้าจำได้แค่ 3 อย่าง

1. **SSF ยกเลิกแล้ว → Thai ESG มาแทน** วงเงิน 300K แยกจาก RMF 500K ใช้ทั้งคู่ได้
2. **ยิ่งรายได้สูง ยิ่งคุ้มมาก** marginal rate 20% = ซื้อ RMF 100K ประหยัด 20K ทันที
3. **RMF ต้องซื้อทุกปี — Thai ESG ไม่บังคับ** รู้ความต่างนี้ก็วางแผนได้ถูก
