**สิ่งที่สองที่น่าสนใจ: ซื้อหุ้น แต่ไม่ได้ตัดสินใจอะไรเลย**

SpaceX มีหุ้น 3 Class:

Class A — 1 vote ต่อหุ้น (สิ่งที่คนทั่วไปซื้อในตลาด)
Class B — 10 votes ต่อหุ้น (Elon เท่านั้น)
Class C — 0 vote ต่อหุ้น (พนักงาน)

Elon ถือหุ้น 42% ของบริษัท แต่คุม voting power 82.4%

แปลว่าถ้าคุณซื้อหุ้น SpaceX ทุกบาท คุณไม่มีสิทธิ์ตัดสินใจอะไรในบริษัทนี้เลย Elon คนเดียวตัดสินทุกอย่าง

ไม่ใช่เรื่องเลวร้าย เพราะ Apple, Google, Meta ก็ออกแบบคล้ายกัน แต่เป็นเรื่องที่ต้องรู้ก่อนตัดสินใจ

**สิ่งที่สาม: บริษัทยังขาดทุนอยู่ แต่ขาดทุนแบบมี plan**

Q1 2026 SpaceX ขาดทุนสุทธิ -$4.28 พันล้านดอลลาร์

แต่เหตุผลคือลงทุนใน Starship — จรวดที่ตั้งใจจะพา payload ไปดาวอังคาร และในอนาคต Elon ตั้งเป้าให้ Starship บินได้วันละ 3 เที่ยว

ในขณะที่ Starlink (ธุรกิจ Internet ดาวเทียม) มีรายได้ $11.4 พันล้านดอลลาร์ คิดเป็น 61% ของรายได้ทั้งหมด และมีผู้ใช้งาน 2.8 ล้านราย

SpaceX ไม่ใช่บริษัทที่ขาดทุนเพราะทำธุรกิจไม่เป็น แต่เพราะกำลัง bet ครั้งใหญ่ที่สุดในประวัติศาสตร์
